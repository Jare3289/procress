คืองี้นะ ตรวจสิบหน่อยสิ ในการแสดง ผลลัพธ์การจำลอง (Simulation Results) อะ มันจะโยกไหลไปตามกัน แต่ที่น่ ๆ ห้องสุดท้ายมันต้องว่างลง เราพิจารณาคะแนนก่อน แล้วจึงจะดูลำดับ เพราะฉะนั้น นักเรียนที่ได้คะแนนน้อยอยู่ท้าย ๆ จะไม่มีโอกาสไปยังห้องที่มีคะแนนสูง จะต้องเข้ามาต่อข้างหลัง เพราะฉะนั้นนักเรียนที่เป็นตัวสำรองที่ไม่ถูกจัดเข้าห้องใดเลย ต้่องได้ไปอยู่ห้องท้ายก่อนเสมอ แล้วย้ายคนที่เป็นตัวจริงในแถวไปก่อนเท่านั้น และทุกห้องต้องเต็ม 40 ถ้าห้างไหนว่าง แล้วไม่มีการเลื่อนตัวจริง ให้เอาตัวสำรองมาต่อห้องสุดท้ายเลย
